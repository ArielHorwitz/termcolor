```
Display terminal colors

Usage: termcolors [OPTIONS]

Options:
  -H, --hues <HUES>                Hue resolution [default: 16]
  -V, --values <VALUES>            Value resolution [default: 4]
  -S, --saturations <SATURATIONS>  Saturation resolution [default: 4]
  -r, --resolution <RESOLUTION>    Resolution (overwrites both value and saturation resolutions)
  -o, --offset <OFFSET>            Offset hue in degrees [default: 0]
  -d, --display <DISPLAY>          Display options [default: rgb] [possible values: rgb, ansi, lum, none]
      --dark <DARK>                Dark color threshold [default: 50]
  -D, --dark-factor <DARK_FACTOR>  Dark color factor [default: 5]
  -l, --legend                     Show legend (hue, saturation, and value)
  -h, --help                       Print help (see more with '--help')
```
